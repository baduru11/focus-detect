# Alarm Sounds
Place these .mp3 files here:
- chime.mp3 — gentle notification chime (Level 1)
- alert.mp3 — urgent alert tone (Level 2)
- siren.mp3 — dramatic siren loop, ~3s (Level 3)
- phase-chime.mp3 — subtle phase transition chime

Until real sound files are sourced, the app falls back to Web Audio oscillator-based sounds.
